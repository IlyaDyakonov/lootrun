# loot.run

Статическая промо-страница **loot.run**: лендинг с колесом удачи, блоками оффера и декоративной вёрсткой на фоновых изображениях. Стили в `styles.css`, тексты с поддержкой локализации через `data-i18n`.

## Карта проекта

```text
lotrun/
├─ index.html      # Основная HTML-структура лендинга
├─ app.js          # Логика интерфейса, интерактив (колесо, UI-сценарии, i18n)
├─ styles.css      # Все стили, адаптив и декоративные элементы
└─ README.md       # Краткое описание и запуск проекта
```

## Запуск локально

```bash
py -m http.server 8000
```

Откройте в браузере: http://localhost:8000/index.html

Или просто открыть в браузере файл index.html
